# Aoshu
PHP自作
## 概要
仮想のグッズ販売の受注管理ができるサイトを作成しました。
管理ユーザーと一般ユーザーにわけ、それぞれログインできるようになっています。
## 使い方
一般ユーザー

・商品マスターの追加・変更、受注の登録・変更

管理ユーザー
